Posted by David Beckett on January 11, 2021

This is the first blog post to kick off the conception of the website. The following will show a quick how to for adding your own blog post.

Making A Blog Post

First, setup the files and folders needed, note the post number that corresponds to the following image (Post1). Create a new file in the same directory named "postx.html" such that x is incremented 1 from the most recent post.

Also create a new folder named "postx" within the "img" folder for any images used in your post

Open index.html and copy the highlighted lines and pastes them just below the comment saying where to post them. Modify the post-title, post-subtitle, and post-meta divs to their new values.

copy and paste the entirety of a previous post into the "postx.html" file.

Navigate to the post heading div shown in the following picture and update the highlighted fields to match those that were put on the index page.

The header image can be changed by updating the url path highlighted in the picture below

Finally, navigate to the post content shown in the image below and fill in the information you want, this post shows some basic formatting for a post. Additional formatting information can be found in "Samplepost.html" which is linked by clicking the below image.
